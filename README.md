# AI-Based CV Analysis System

This project is a web-based application that analyses CVs using a combination of rule-based scoring and AI-generated feedback.

## Features
- CV parsing (PDF & DOCX)
- Rule-based scoring (structure, skills, formatting, content)
- AI feedback using Claude API
- Bilingual support (English & Uzbek)

## Tech Stack
- Python (Flask)
- SQLAlchemy
- Flask-Babel
- Claude API

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   python app.py
